Zplot.org

Welcome to zplot.org, home of the little graphing tool called zplot. Zplot is written in pure Python (based on an earlier version in Tcl) and can readily create a wide range of graphs in EPS or SVG formats.

Learn more by reading the overview, browsing examples, or by checking out the documentation.

The easiest way to install zplot: 

pip install zplot
You can also grab it from github.
